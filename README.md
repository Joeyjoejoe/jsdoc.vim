## JSDoc Vim 
> Generate JSDoc block comments in vim.

## Install

### Vundle
> Vundle is the recommended installation method

Add to your bundle list:

```
Bundle 'mxw/vim-jsx'
```

### Pathogen

Clone into bundle directory
```
$ cd ~/.vim/bundle
```
```
$ git clone https://github.com/joegesualdo/jsdoc.vim
```
